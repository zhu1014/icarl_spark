# SPARK: Simple and Parameter-Free Knowledge Graph Embedding for Class Incremental Learning

### Get started

For single gpu run
```Shell
python main.py --dataset cifar100 --epochs 200 --lr 0.3 --device 0
```

### Environment

- Python 3.8
- Pytorch 2.2.0
- CUDA 12.0

### Requirement

+ See `requirements.txt` for environment.
